# CONTENT-SPEC — 学习页内容规范

> 每个学习主题页（`<topic>/index.html`）的结构规范。
> 新页从 `_template/index.html` 复制起步——工程层（token、quiz 引擎、主题切换、复制按钮）已内置，不用背；
> 本文档管「写什么、什么必须一致、什么自由」。发布前跑 `CLAUDE.md` 里的检查清单。
> 2026-07-07 由四页对照审计 + 逐项拍板产出（审计与拍板记录见 Obsidian 项目档案）。

## 0. 目的与读者（所有取舍的裁判）

一页的目标读者是 **3 个月后的自己**：30 分钟内能重走一遍这张地图，并通过 quiz。
任何风格之争用这条裁决：帮助「重走 + 通过」的进冻结层；无关的归自由层。

## 1. 冻结层（每页必须一致，改动 = 改本文档 + 改模板）

### 1.1 页面骨架（顺序固定）

```
hero（eyebrow → h1 → thesis → credit → legend 页内导航；主题切换按钮在右上角）
→ §1 心智模型（第一节必须是心智模型）
→ §2 … §N-2 主题正文（形态自由，见第 2 节）
→ §N-1 来源书架
→ §N 自测 quiz（永远是最后一节）
→ footer（首行「← 返回学习主页」，相对路径 ../）
```

### 1.2 命名

- `<title>` 与主页卡片 h2 同格式：`中文标题 · English Name`。
- eyebrow 从受控词表选：`Field Guide`（默认）/ `Field Manual` / `Critical Read`。要新词先加进本表。
- 章节编号 `§1…§N`，无 emoji；称呼读者用「你」。

### 1.3 来源书架

- 固定为倒数第二节。每条 = 来源名 + 一句「这家独有、别家没有的东西」+ 外链。
- **一手来源必须带日期**（发表日期；拿不到就标「核对于 YYYY-MM」）。
- 正文内联链接不限制，但书架必须收全本页引用过的一手来源。

### 1.4 单一事实来源

- 会变的数字只允许存在一处。quiz 题数只写在 JS 的 `QS` 数组里，
  页内导航和计分条由 JS 推导填充，HTML 与小节导语不写死题数。
- 主页卡片描述里的数字，发布前与页面事实核对（检查清单有此项）。

### 1.5 链接

- 站内一律相对路径（`../unknowns/#t11`、`demos/01-x.html`），禁止 `https://zbisure.github.io/...` 绝对 URL——绝对 URL 会弄断本地预览。
- 正文提及站内其他学习页，必须给真链接，不许只提名字。
- 「姊妹篇」关系必须双向互链（footer 各放一条）。

### 1.6 quiz 行为（引擎在模板里，别重写）

全对才通过；答错给内联讲解 + 「重读 §X →」锚点；未通过可重置重考；
通过后给「落地清单」（把学到的变成下一步动作）。

### 1.7 翻译

- 界面文案一律中文，复制按钮文案「复制 → 已复制 ✓」。
- **外来英文 prompt**：保留英文原文 + 「中文大意」对照——prompt 是拿来用的，不是拿来读的。
- **自产中文模板**（CLAUDE.md 片段、反借口表、ADR 模板等）：直接中文，不需要英文对照。

### 1.8 工程基线（模板已内置）

- 自包含单文件：无外部 CDN、无外部字体，内联 CSS/JS。
- CSS token 全集 + 明暗双主题：跟随系统（`prefers-color-scheme`）+ 右上角手动切换（`data-theme`，localStorage 记忆，键 `zb-theme`）。
- 点选揭示交互用 `<button>` + `aria-pressed`；`prefers-reduced-motion` 降级。
- 视觉语言：地形图/野外手册——纸面底色、等高线装饰、测绘蓝强调色。

## 2. 自由层（随主题发挥，不需要页页一致）

- 章节数量与标题；正文组件形态（四象限、决策卡、时间线、判定 chip……鼓励每页发明自己的词汇）。
- **章节目录（TOC）：可选**。章节多（≥6）时建议加；左侧固定位置，可收起；用模板里的 `.toc` 样式；章节少时可省略。移动端（≤900px）自动隐藏。
- **可复制模板区：可选**。有就用模板里的 `.copy` 规范；没有也合规。
- 子资源目录（如 `unknowns/demos/`）。
- quiz 题数（8–12 题为宜）与通关文案。

## 3. 非目标（决定不做，不是忘了做；想做先改这里）

- OG / social meta
- quiz 进度持久化（localStorage 只用于主题偏好）
- print 样式
- 评论、访问统计、任何第三方脚本

## 4. 站点级不变量（写单页时看不见，发布前核对）

- 主页卡片：编号递增、`YYYY-MM`、标签、标题（中 · 英）、描述与页面事实一致。
- 主页 hero 不承诺「每个主题都有可复制模板」（模板是自由层）。
- 根 `index.html` 的基础 token 与模板同源。
