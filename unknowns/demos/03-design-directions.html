四个设计方向

当你说不清自己想要什么设计时，就别硬去描述 —— 让 AI 一次做出四个互不兼容的方向、直接渲染出来， 让直觉替你写规格。对着现成的东西提反应，远比凭空想象容易。

提示词
“I want a review-queue dashboard for Acme but I have no visual taste and don't know what's possible. Make me one HTML page with 4 wildly different design directions so I can react to them.”

中文大意：我想给 Acme 做一个审阅队列仪表盘，但我没什么视觉品味，也不知道有哪些可能性。请做一个 HTML 页面，里面放 4 个风格迥异的设计方向，让我可以对着它们提反馈。

Claude 的产出

同一份数据，四种设计哲学。下面每个界面渲染的都是完全相同的审阅队列 —— Acme 里五条待审视频 —— 唯一的变量就是设计本身。在每个界面下方， 点一点你想偷师（steal）或跳过（skip）的细节标签； 你的选择会在页面底部自动拼成一段英文回复。

如何反馈

回复一个方向，加上你要偷师的细节

你不需要写设计说明书 —— 这正是重点所在。挑出气质对路的那个方向， 再从其他方向里点名两三个值得搬过来的细节。 点选一个方向、再点上面的小标签，回复就自动写好了。 


      
    

    

      把这段英文回复原样贴回对话即可。Claude 会把你偷师的细节揉进胜出的方向，
      回给你一版打磨过的设计 —— 这可比费劲描述你自己都没意识到的品味，省话得多。
